# Excel-data-load
Excel-data-load

### Prerequisite
    Node version 14

### Run Local
    yarn install
    yarn build
    npm install
	npm run build:prod
	npm run dev

### Publish
	npm run build:prod

### Data Load
Add EXCEL_FILE to data Folder File Must Be In .xlsx Format.
### File Name
Tenant+Configurations.xlsx
### Add Owner name/id
 Ex:-"2a72161a-ccf9-403d-86a7-d1914a7f660a"
